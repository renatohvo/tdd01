## Como executar o projeto

Pré-requisitos: Java 11

```bash
# clonar repositório
git clone https://github.com/renatohvo/tdd01.git

# entrar na pasta do projeto backend
cd backend

# executar o projeto
./mvnw spring-boot:run
```
